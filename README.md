# DotGameVideo
⚡ Retro terminal game with dual modes: tap matching colored dots before time runs out! Classic mode lets you choose, Color Switch keeps you guessing. Features CRT scanlines, haptic feedback, separate leaderboards &amp; authentic green-screen nostalgia. iOS/iPadOS 💚
